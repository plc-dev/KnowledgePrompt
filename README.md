# KnowledgePrompt

## Installation and updating

Use pip to install like below. Rerun the command to check for and install updates.

```
pip install git+https://github.com/plc-dev/KnowledgePrompt
```

## Usage

Features:

-   OPT-model-wrapper
-   prompt-generator
-   example sampling

Example:

```{python}
import prompt_tools
from prompt_tools import OPTRunner
```
